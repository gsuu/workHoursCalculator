# 근무시간 계산기

구글 스프레드시트 기반 계산기를 실제 Vue 작업환경으로 옮긴 프로젝트입니다.
지금은 `Vite + Vue` 기반 개발 환경에서 실행하고 빌드할 수 있습니다.

## 현재 구조

- `Vite` 개발 서버
- `Vue 3` 단일 앱
- 계산 로직은 `src/App.vue` 한 파일에 최대한 모음
- DB 없음

제 의견은 분명합니다.
이 프로젝트는 화면이 하나이고 계산 규칙이 핵심이라, 라우터나 상태관리 라이브러리까지 넣는 건 과합니다.
그래서 Vue 환경만 제대로 만들고 책임은 `App.vue` 중심으로 유지했습니다.

## 실행

```powershell
npm.cmd install
npm.cmd run dev
```

기본 개발 서버는 보통 아래 주소로 열립니다.

- `http://localhost:5173`

## 빌드

```powershell
npm.cmd run build
```

빌드 결과물은 `dist` 폴더에 생성됩니다.

## 계산 기준

- 평일 근무
  - 1일 8시간 또는 1주 40시간 초과분은 연장근로
  - 22시~06시는 야간근로
- 휴일 근무
  - 8시간 이내는 1.5배
  - 8시간 초과는 2.0배
  - 야간은 별도 가산으로 최대 2.5배
- 휴무일 근무
  - 휴일근로가 아니라 연장근로 기준

## 공휴일 자동 판정

- 일요일 또는 등록된 대한민국 공휴일: `휴일`
- 토요일: `휴무일`
- 그 외: `평일`

공휴일 데이터는 코드에 내장되어 있고, 2025~2027 일정이 반영되어 있습니다.

## Windows 한글

파일은 UTF-8 기준입니다.
PowerShell 콘솔에서만 한글이 깨져 보이면 코드페이지 문제일 수 있으니 `chcp 65001`로 확인하는 편이 안전합니다.
